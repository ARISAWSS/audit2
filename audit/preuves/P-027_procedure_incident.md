## Procédure de gestion des incidents de sécurité — Clinique Santé Plus

### 1. Détection
- **Sources** : Alertes EDR, utilisateurs, prestataire IT
- **Responsable détection** : Référent SI / Prestataire IT

### 2. Analyse initiale
- **Responsable** : Référent SI
- **Délai** : Immédiat
- **Actions** : Isoler système affecté, collecter logs

### 3. Escalade
- **Niveau 1** : Référent SI
- **Niveau 2** : Direction
- **Niveau 3** : Prestataire IT (support expert)

### 4. Communication
- **Interne** : Direction, IT
- **Externe** : Prestataire IT, autorités (si nécessaire)

### 5. Résolution
- **Responsable** : Prestataire IT (avec validation Direction)
- **Documentation** : PV incident

### Observations / Manques identifiés
- **Délai CNIL** : Non mentionné explicitement (72h)
- **Communication patients** : Procédure non détaillée
- **Kit de crise** : Non documenté (contacts urgents, modèles)
- **RACI détaillé** : Rôles partiellement définis
