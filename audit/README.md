# Audit SSI — Clinique Santé Plus

Projet académique — Module "Audit et gestion des risques — Normes ISO"  
Efrei Paris — Période : 1er décembre 2025 — 5 janvier 2026

## Description

Audit de sécurité des systèmes d'information réalisé pour la Clinique Santé Plus (organisation fictive).

## Structure

- `livrables/` : Documents de livraison (lettre de mission, note de cadrage, programme d'audit, journal de bord, registre de preuves, fiches constats, rapport d'audit)
- `preuves/` : Preuves collectées lors de l'audit (30 preuves anonymisées)

## Résultats

- 7 constats identifiés
- 30 preuves collectées
- 7 recommandations priorisées

## Référentiels

- ISO/IEC 27001/27002
- RGPD
- HDS
- ANSSI
